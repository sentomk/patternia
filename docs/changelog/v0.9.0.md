# Patternia v0.9.0 Release Note

**Release Date:** March 13, 2026
**Version:** 0.9.0

---

## Overview

Patternia v0.9.0 is a distribution and packaging release.

It adds first-class vcpkg support so the library can be consumed through
`vcpkg install patternia` without any manual integration steps. No API
changes were made. The public surface from v0.8.5 is preserved as-is.

---

## Highlights

### vcpkg Support

Patternia is now available through the vcpkg package manager.

```bash
vcpkg install patternia
```

Or via `vcpkg.json` manifest:

```json
{
  "dependencies": ["patternia"]
}
```

CMake projects that use the vcpkg toolchain file can consume the library
through the same target name as before:

```cmake
find_package(patternia CONFIG REQUIRED)
target_link_libraries(your_target PRIVATE patternia::patternia)
```

### Installation Documentation

The installation guide now covers all supported integration methods:

- `vcpkg install`
- `FetchContent`
- `find_package` after CMake install
- Git submodule
- Direct header copy

---

## API Compatibility

No breaking changes. All code that builds against v0.8.5 builds unchanged
against v0.9.0.
